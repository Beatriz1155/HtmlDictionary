HTML ESTRUTURA

Como foi dito antes, Html é uma linguagem de marcação utilizada na construção de páginas Web.
Aqui está um exemplo:

Explicação do exemplo:

!DOCTYPE html - define que este documento, é um documento HTML5;

HTML - é o elemento raiz de uma página HTML;

Head - define o cabeçalho de um documento HTML;

Title - define o título da página, que é exibido na barra de título dos navegadores;

Body - define o conteúdo principal, o corpo do documento. Esta é a parte do documento HTML que é exibida no navegador. No corpo podem-se definir atributos comuns a toda a página,
como cor de fundo, margens, e outras formatações;

O que é um elemento HTML?

Um elemento HTML é definido por uma tag de inicio, o conteudo, e uma tag de fim.

Alguns elementos HTML não possuem conteúdo (como o elemento br, uma tag de texto). Esses elementos são chamados de elementos vazios.
Elementos vazios não possuem uma tag final.

META:

O Meta define propriedades da página, como codificação de caracteres, descrição da página, autor, etc.

LINK:

Os links HTML são definidos com a tag a.
O destino do link é especificado no atributo href.

STYLE:

O atributo style é usado para adicionar estilos a um elemento, como cor, fonte, tamanho e muito mais.
