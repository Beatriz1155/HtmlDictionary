HTML TEXTO

P:

A tag p define um parágrafo.

H1 ATÉ H6:

As tags h1 até h6 são usadas para definir cabeçalhos HTML.
H1 define o cabeçalho mais importante e a H6 define o título menos importante.

STRONG:

A tag strong é usada para definir o texto com forte importância. O conteúdo interno geralmente é exibido em negrito .

B:

A tag b especifica o texto em negrito sem nenhuma importância extra.

I:

A tag i define uma parte do texto em uma voz ou humor alternativo, é frequentemente usada para indicar um termo técnico, uma frase de outro idioma, um pensamento, um nome de navio, etc.
O conteúdo interno geralmente é exibido em itálico. Use o elemento i somente quando não houver um elemento semântico mais apropriado, como:

Em - texto enfatizado;
Strong - texto importante;
Mark - texto marcado/destacado;
Cite - o título de um trabalho;
Dfn - um termo de definição.

U:

A tag u representa algum texto não articulado e com estilo diferente do texto normal, como palavras com erros ortográficos ou nomes próprios em texto chinês.
O conteúdo interno geralmente é exibido com um sublinhado.

MARK:

A tag mark define o texto que deve ser marcado ou destacado.

BR:

A tag br insere uma única quebra de linha.

HR:

A tag hr define uma quebra temática em uma página HTML (por exemplo, uma mudança de tópico), geralmente é exibido como uma régua horizontal usada para separar o conteúdo
(ou definir uma alteração) em uma página HTML.
