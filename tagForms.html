HTML FORMS

FORM:

O elemento HTML form é usado para criar um formulário HTML para entrada do usuário.
O elemento form é um contêiner para diferentes tipos de elementos de entrada, como: campos de texto, caixas de seleção, botões de opção, botões de envio, etc.

INPUT:

O elemento HTML input é o elemento de formulário mais usado.
Um input pode ser exibido de várias maneiras, dependendo do type atributo.
Aqui estão alguns exemplos:

TXTAREA:

O input type="text" define um campo de entrada de linha única para entrada de texto.

SELECT:

O input type="checkbox" define uma caixa de seleção.
As caixas de seleção permitem que um usuário selecione ZERO ou MAIS opções de um número limitado de opções.

OPTION:

A tag option define uma opção em uma lista de seleção, este elemento entra em elementos como o select, o optgroup ou datalist.
A tag option pode ser usada sem nenhum atributo, mas geralmente é necessário o atributo value, que indica o que é enviado ao servidor no envio do formulário.

OPGROUND:

A tag opygroup é usada para agrupar opções relacionadas em um elemento select (lista suspensa). Se tiver uma longa lista de opções, os grupos de opções relacionadas serão mais fáceis de manipular para um usuário.

LABEL:

A tag label define um rótulo para muitos elementos de formulário, o elemento é útil para usuários de leitores de tela, porque o leitor de tela lerá o rótulo em voz alta quando o usuário focar no elemento.
O elemento label também ajuda os usuários que têm dificuldade em clicar em regiões muito pequenas (como botões de opção ou caixas de seleção) - porque quando o usuário clica no texto dentro
do elemento label, ele alterna a entrada, aumentando a área de acerto.

BUTTON:

A tag input type="radio" define um botão de opção.
Os botões de opção permitem que um usuário selecione UMA opção dentre um número limitado de opções.

FIELDSET:

A tag fieldset é usada para agrupar elementos relacionados em um formulário, ela desenha uma caixa ao redor dos elementos relacionados.

LEGEND:

A tag legend define uma legenda para o elemento fieldset.
