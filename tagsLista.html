HTML LISTAS

UL:

A tag ul define uma lista não ordenada (marcada).
Usa a tag ul junto com a tag li para criar listas não ordenadas.

IL:

A tag li define um item de lista.
A tag li é usada dentro de listas ordenadas (ol), listas não ordenadas (ul) e em listas de menu menu.

OL:

A tag HTML ol define uma lista ordenada.
Uma lista ordenada pode ser numérica ou alfabética, começando com a tag ol e cada item da lista começa com a tag li tag.

O type, atributo da tag ol, define o tipo de marcador do item da lista:

DL:

A tag dl define uma lista de descrição.
A tag dl é usada em conjunto com a tag dt (define termos/nomes) e a tag dd (descreve cada termo/nome).
