HTML SCRIPTING

SCRIPT:

O elemento script contém instruções de script ou aponta para o arquivo de script externo por meio do artributo scr.
Geralmente é usado em JavaSript para a manipulação de imagem, validação de formulários e alteração dinâmica de conteúdos.

NOSCRIPT:

A tag noscript define um conteúdo alternativo a ser exibido para usuários que desativaram scripts em seus navegadores ou possuem um navegador que não oferece suporte a scripts.
O elemento noscript pode ser usado nas tags head e body. Quando usado dentro da tag head, o elemento noscript pode conter apenas os elementos link, style e meta.
