HTML TABELA

TABLE:

Uma tabela em HTML é defenida pela tag table, e consiste em células de tabela dentro de linhas e colunas.

- Cada célula da tabela é definida por uma tag td.
- Td significa dados da tabela, e tudo o que fica entre td é o conteúdo da célula da tabela.

- Se quiser fazer linhas na tabela, use a tag tr.
- Se deseja que as células sejam células de cabeçalho da tabela. Neste caso, use a tag th em vez da tag td, que significa cabeçalho da tabela.

CAPTION:

A tag caption define uma legenda de tabela, e deve ser inserida imediatamente após a tag table.

COLGROUP:

A tag colgroup especifica um grupo de uma ou mais colunas em uma tabela para formatação, ela é útil para aplicar estilos a colunas inteiras, em vez de repetir os estilos para cada célula, para cada linha.
A tag col especifica as propriedades da coluna para cada coluna dentro de um elemento colgroup, como por exemplo a cor de fundo de três colunas.

THEAD:

A tag thead é usada para agrupar o conteúdo do cabeçalho em uma tabela HTML.
O elemento thead é usado em conjunto com os elementos tbody e tfoot para especificar cada parte de uma tabela (cabeçalho, corpo, rodapé).

- A tag tbody é usada para agrupar o conteúdo do corpo em uma tabela HTML.
- A tag tfoot é usada para agrupar o conteúdo do rodapé em uma tabela HTML.
- Os elementos thead, tbody e tfoot devem ter uma ou mais tags tr dentro.
